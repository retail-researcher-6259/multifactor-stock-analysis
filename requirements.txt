# Installation Notes:
# ==============================================================================
# 1. Install TA-Lib C library first (required for ta-lib Python package):
#    - Windows: Download from https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib
#    - macOS: brew install ta-lib
#    - Linux: sudo apt-get install ta-lib
#
# 2. For TensorFlow GPU support (optional):
#    - Install CUDA and cuDNN following TensorFlow documentation
#    - Replace tensorflow with tensorflow-gpu
#
# 3. Create virtual environment (recommended):
#    python -m venv venv
#    venv\Scripts\activate  # Windows
#    source venv/bin/activate  # macOS/Linux
#    pip install -r requirements.txt
#
# 4. For development/testing only, install optional packages:
#    pip install pytest pytest-cov black pylint
#
# 5. If you encounter issues with curl-cffi:
#    pip install --upgrade curl-cffi
#
# 6. For lighter installation without deep learning:
#    Comment out tensorflow line if not using LSTM/Transformer models
#
